# Campus-Networking
1. LAN: First of all, the LANs were connected and configured.
2. WAN: After succesful LAN connectivity, WAN was configured i.e. routers were interconnected with each other.
3. OSPF: All the sunetworks connected to different routers were made able to communicated with each other using star-ring hybrid topology.
4. Static Routing: All routers can send packets to all other routers thus all hosts to all hosts.
5. DHCP: Routers act as DHCP servers and automatically assign IPs removing the need the of exteral severs.
